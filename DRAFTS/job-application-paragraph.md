# Job Application Paragraph

I recently built and validated JMOA, a build-time JVM optimization project for Spring Boot services. The work includes bytecode rewriting, adapter consolidation, Spring Boot fat-JAR/exploded-Boot materialization, runtime-origin verification, and container-level memory measurement using smaps PSS, Private_Dirty, cgroup `memory.current`, and NMT. I validated it across three services, including a public Spring PetClinic no-CDS case with a confirmed ~4.6 MB median PSS reduction. I would be excited to bring this kind of JVM/runtime performance work to your team.
